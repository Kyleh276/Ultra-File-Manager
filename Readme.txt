This Program is meant to make it easier to organize files in a dierctory.
It has a default setting and a custom setting for filetypes to include 
in the operation(s).
Larger file sizes will take longer than small filesizes.
The Github Page is: https://github.com/JiinxC/Ultra-File-Manager.git
Everything that throws errors has error codes.
There is also an .exe file of the entire runnable program.
